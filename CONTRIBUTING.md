# Contributing to Awesome Retro Computing

Thanks for your interest in contributing! This is a community-curated list, and we welcome additions from anyone who loves retro computing.

## How to Contribute

1. **Fork** this repository
2. **Create a branch** for your changes (`git checkout -b add-my-resource`)
3. **Add your entry** following the format below
4. **Submit a Pull Request**

## Adding a New Entry

Each entry should follow this format:

```markdown
* [Resource Name](https://url) — _"Brief description of the resource"_
```

### Guidelines

- **Relevant**: Must be related to retro/vintage computing
- **Working links**: Verify the URL works before submitting
- **No duplicates**: Check the list first
- **Alphabetical**: Add entries in alphabetical order within their section
- **Description**: Include a brief, quoted description
- **Categories**: Place entries in the most appropriate existing category

## Adding a New Category

If your resource doesn't fit an existing category:

1. Add a new subsection under the most relevant parent section
2. Use `###` or `####` heading level consistent with nearby sections
3. Include at least 2-3 entries to justify a new category

## What We're Looking For

- Emulators and simulators
- Hardware preservation projects
- Retro computing blogs, podcasts, and communities
- Classic operating systems and software
- Development tools for vintage platforms
- Museums and archives
- Hardware vendors and repair resources

## What We're NOT Looking For

- Piracy or ROM download sites
- Broken or dead links
- Commercial products with no retro relevance
- Duplicate entries
