# Contributing to Awesome Retro Computing

Thank you for contributing to the Awesome Retro Computing list!

## Adding Projects

1. **Check the list** - Make sure your project isn't already included
2. **Follow the format** - Use the existing category structure
3. **Add meaningful descriptions** - Explain what makes the project notable
4. **Keep it relevant** - Projects should relate to retro computing

## Categories

- Vintage Hardware
- Emulators
- Operating Systems
- Programming Languages
- Games
- Preservations Efforts

## Pull Request Process

1. Fork the repo
2. Add your project to the appropriate category in README.md
3. Submit PR with brief description

## Quality Standards

- Active or notable projects only
- Include working links
- Brief description (1-2 sentences)
- Not commercial products
